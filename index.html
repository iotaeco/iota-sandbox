IOTA Sandbox

The code that will run in this sandbox is in src/index.js. As you make changes to that file you should see the output in the console window update.

The code is already configured with curl.lib.js to perform proof of work using WebGL in the browser.

Open the console with the Show Console button above to see the output.

The original code for the sandbox can be found on GitHub https://github.com/iotaeco/iota-sandbox
